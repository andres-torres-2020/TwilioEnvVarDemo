A Node.js demo app to invoke Twilio Environment and Variable APIs

### Accessing env vars in node.js application:
Files:
- config.js
- development.env
- package.json

# [demo app]
## env-demo.js
 demo for Twilio Environment & Variable classes

### setup:
```
  mkdir TwilioEnvVarDemo
  cd TwilioEnvVarDemo
  npm init
  npm i dotenv
  npm i twilio
```
###  how to run this demo:
```
npm run dev
```
